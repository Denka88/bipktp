О приложении

Общее описание

Приложение представляет собой систему для управления календарно-тематическими планами (КТП) учебного процесса. Оно позволяет преподавателям создавать, редактировать и структурировать учебные планы, а администраторам - управлять справочной информацией и пользователями системы.

Основные возможности

  • Создание и редактирование календарно-тематических планов
  • Структурирование КТП по разделам и занятиям
  • Генерация документов КТП в формате Word
  • Управление справочниками (комиссии, дисциплины, периоды и др.)
  • Разграничение прав доступа (администраторы и преподаватели)
  • Постраничное отображение списков с пагинацией

Технологии

Backend:
  • Java 24
  • Spring Boot 3.5.x
  • Spring Security
  • Spring Data JPA
  • Apache POI (генерация документов)
Frontend:
  • Thymeleaf
  • Bootstrap 5
  • JavaScript
  • Bootstrap Icons
